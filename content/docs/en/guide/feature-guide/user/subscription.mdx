Subscriptions are periodic packages. After purchase, you enjoy the included quota or privileges for the duration of the plan's cycle. Open **Subscription** in the left sidebar of the AI Router console, or visit `/console/subscription` directly.

<Callout type="warn">
AI Router is a brand-new demo. The concrete plans, prices, cycles, included-quota amounts, and payment methods shown here are **not finalized yet** — every amount below is a placeholder marked *(TBD / 待补)*. Do not treat any figure as a real offer.
</Callout>

The subscription list displays the available packages. For each package you would typically see:

- **Plan name** — a label for the package.
- **Price** — *(pricing TBD / 价格待补)*.
- **Billing cycle / validity** — e.g. daily, weekly, or monthly *(cycles TBD / 周期待补)*.
- **Included quota** — the amount of usage the plan grants for the cycle *(quota amounts TBD / 额度待补)*.

Conceptually, a subscription bundles a recurring quota that resets or applies each cycle, as an alternative to topping up a pay-as-you-go balance. Both ultimately fund the same thing: calls you make through your API key against the OpenAI-compatible endpoint at `https://ai.sammier.com/v1`.

## View Current Subscription Status

Once a plan is active, the top of the subscription page shows your current subscription details:

- **Plan name and expiration date** — when the current cycle ends.
- **Remaining quota** — how much of the plan's included quota is left for the cycle.
- **Auto-renewal toggle** — choose whether the plan renews automatically when it expires.

## How quota is consumed

A subscription's included quota is spent the same way as a regular balance: each call to a model group draws down quota according to that model's rate. The exact per-model rates are **TBD / 费率待补** for this demo. To inspect what each request actually consumed, use the usage and logs views in the console.
