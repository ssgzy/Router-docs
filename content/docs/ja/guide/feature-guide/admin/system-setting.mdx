Root 専用のグローバル設定センターです。サイト情報、課金ルール、支払い設定など、すべてのシステムレベルのパラメータを管理します。Root アカウントでログイン後、左側ナビゲーションの「設定」をクリックするか、`/console/setting` に直接アクセスしてください。

システム設定ページには一般、課金、支払い、メール、セキュリティ、OAuth などすべてのシステムレベル設定を含む複数のタブがあります。

## 一般設定

システム設定ページの「一般設定」タブをクリックしてください。

サイト名、ホームページのお知らせ、ドキュメントリンク（左側ナビゲーションの「ドキュメント」ボタンのリンク先 URL）を設定できます。

<Callout type="info">
  「ドキュメント URL」フィールドに入力すると、ホームページの左側ナビゲーションに「ドキュメント」ボタンが表示され、そのリンク先に移動します。
</Callout>

## 課金・倍率設定

「課金設定」または「倍率設定」タブをクリックしてください。

倍率設定で各モデルの実際の課金価格を決定します。グループ倍率でユーザーセグメントごとに差別化した価格設定が可能です。

## 登録・セキュリティ設定

「セキュリティ設定」または「登録設定」タブをクリックしてください。

公開登録の開放可否を制御し、悪意ある登録を防ぐ Turnstile CAPTCHA を有効化できます。

## OAuth 設定

「OAuth 設定」タブをクリックしてください。

サードパーティログインに必要な OAuth アプリケーション認証情報を設定します。設定後、ユーザーは対応するプラットフォームアカウントでログインできるようになります。
