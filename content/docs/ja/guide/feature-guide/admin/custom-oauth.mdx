title: カスタム OAuth プロバイダー
description: OIDC 準拠のカスタムログイン方法を追加
---

組み込みの OAuth プロバイダーに加えて、Root は OIDC 準拠のカスタムログイン方法を追加できます。Root アカウントでログイン後、システム設定（`/console/setting`）に移動して「カスタム OAuth」セクションを見つけてください。

## カスタム OAuth プロバイダーの追加

1. 「プロバイダーを追加」をクリックすると設定ダイアログが表示されます

2. 「Discovery URL」フィールドに OIDC プロバイダーの Discovery アドレス（通常 `/.well-known/openid-configuration` で終わる）を入力し、「自動検出」をクリックすると、システムが Authorization Endpoint、Token Endpoint などの設定を自動入力します

3. Client ID と Client Secret を入力します（OAuth プロバイダーのアプリ管理ページから取得）
4. 表示名を入力します（ログインページでユーザーに表示されるボタンのテキスト）
5. 「保存」をクリックすると、ユーザーのログインページに新しいログインオプションが表示されます
