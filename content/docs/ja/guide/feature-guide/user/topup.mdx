title: クォータとチャージ

クォータはプラットフォーム内部の課金単位です。消費量 = 実際のトークン数 × モデル倍率。左側ナビゲーションの「チャージ」をクリックするか、`/console/topup` に直接アクセスしてください。

![チャージページ](/assets/guide/feature-guide/topup.png)

オンライン決済と換金コードの 2 つの方法でアカウントにクォータを追加できます。

| 方法 | 説明 |
| --- | --- |
| 換金コード | 管理者が生成したコードを入力してクォータを直接追加 |
| EPay | 国内決済（Alipay、WeChat Pay など） |
| Stripe | 国際クレジットカード決済 |
| Creem / Waffo | 国際決済プラットフォーム |

## オンライン決済チャージ

1. チャージページでチャージ金額を選択するか、カスタム金額を手動で入力します
2. 支払い方法を選択します（EPay / Stripe / Creem / Waffo）
3. 「チャージ」ボタンをクリックすると、対応する決済プラットフォームにリダイレクトされます

4. 決済プラットフォームで支払いを完了すると自動的に戻り、アカウント残高が更新されます

## 換金コードチャージ

1. チャージページで「換金コード」入力エリアを見つけます
2. 管理者から提供された換金コードを貼り付けるか入力します

3. 「換金」ボタンをクリックすると、システムがコードを検証します
4. 換金成功後、ページに獲得したクォータ量が表示され、アカウント残高が更新されます

## 紹介報酬

各アカウントには固有の紹介コードがあります。紹介した人が購入すると報酬クォータが得られます。

1. チャージページまたは個人設定ページで「紹介」エリアを見つけ、専用の紹介コードをコピーします

2. 紹介コードを他の人に共有します。相手は登録時にコードを入力します
3. 紹介されたユーザーが購入すると、報酬クォータ残高が自動的に増加します
4. 「残高に振り替え」ボタンをクリックして報酬クォータをメインアカウント残高に移すと使用できます
