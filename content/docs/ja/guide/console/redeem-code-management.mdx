title: 引き換えコード
---

import { Callout } from 'fumadocs-ui/components/callout';

ここで Code Router の引き換えコードを管理できます。引き換えコードは、内部テスト、承認済み顧客への提供、イベントでの贈呈、または経理調整にのみ使用されます。

<Callout type="warn">
  引き換えコードの使用は、認可範囲、プラットフォームの規則、および現地の法令要件に準拠する必要があります。
</Callout>
