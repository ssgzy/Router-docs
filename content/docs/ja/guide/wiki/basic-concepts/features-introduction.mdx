1. 🎨 新しいUIインターフェース（一部のインターフェースはまだ更新が必要です）
2. 🌍 多言語対応（改善予定）
3. 🎨 [Midjourney-Proxy(Plus)](https://github.com/novicezk/midjourney-proxy)インターフェースのサポートを追加（使用前に、サードパーティプロジェクトおよびアップストリームサービスのライセンス、コンテンツセキュリティ、およびサービス利用規約を確認する必要があります）
4. 💰 合法にライセンスされたデプロイメントにおける内部残高、コスト配分、または企業顧客の会計管理をサポートし、システム設定で設定できます：
   - [x] 易支付
5. 🔍 合法にライセンスされたチャネルの使用量または残高情報の照会をサポート：
6. 📑 ページネーションで1ページあたりの表示数を選択可能
7. 🔄 SQLiteデータベースストレージをサポートし、すぐに使用でき、軽量で便利です
8. 💵 組織内のコスト計算または企業顧客の会計測定をサポートし、システム設定-運用設定 で設定できます
9. ⚖️ チャネルの **加重ランダム** をサポート
10. 📈 データダッシュボード（コンソール）
11. 🔒 トークンが呼び出せるモデルを設定可能
12. 🤖 Telegram認証ログインをサポート：
    1. システム設定-ログイン登録設定-Telegram経由でのログインを許可
    2. [@Botfather](https://t.me/botfather)にコマンド /setdomain を入力
    3. あなたのボットを選択し、http(s)://あなたのウェブサイトアドレス/login を入力
    4. Telegram Bot 名は、ボットのユーザー名から @ を除いた文字列です
13. 🎵 [Suno API](https://github.com/Suno-API/Suno-API)インターフェースのサポートを追加（使用前に、サードパーティプロジェクトおよびアップストリームサービスのライセンス、コンテンツセキュリティ、およびサービス利用規約を確認する必要があります）
14. 🔄 Rerankモデルをサポートし、現在CohereとJinaと互換性があり、Difyに接続可能です
15. ⚡ **[OpenAI Realtime API](https://platform.openai.com/docs/guides/realtime/integration)** - OpenAIのRealtime APIをサポートし、Azureチャネルをサポート
16. ルーティング /chat2link を使用してチャットインターフェースに入ることをサポート
17. 🧠 モデル名のサフィックスで推論努力（reasoning effort）を設定することをサポート：
    1. OpenAI oシリーズモデル
       - サフィックス `-high` を追加して high reasoning effort に設定（例: `o3-mini-high`）
       - サフィックス `-medium` を追加して medium reasoning effort に設定（例: `o3-mini-medium`）
       - サフィックス `-low` を追加して low reasoning effort に設定（例: `o3-mini-low`）
    2. Claude 思考モデル
       - サフィックス `-thinking` を追加して思考モードを有効にする（例: `claude-3-7-sonnet-20250219-thinking`）
18. 🔄 思考からコンテンツへの変換、`チャネル-編集-チャネル追加設定` で `thinking_to_content` オプションを設定することをサポートし、デフォルトは`false`です。有効にすると、思考内容 `reasoning_content` が `<think>` タグに変換され、コンテンツに連結されて返されます。
19. 🔄 モデルレート制限、`システム設定-レート制限設定` でモデルレート制限を設定することをサポートし、総リクエスト数制限と成功リクエスト数制限の設定をサポートします
20. 💰 キャッシュ課金サポート、有効にするとキャッシュヒット時に設定された比率で課金できます：
    1. `システム設定-運用設定` で プロンプトキャッシュ倍率 オプションを設定
    2. チャネルで プロンプトキャッシュ倍率 を設定します。範囲は 0-1 で、例えば 0.5 に設定するとキャッシュヒット時に 50% で課金されます
    3. サポートされるチャネル：
    - [x] OpenAI
    - [x] Azure
    - [x] DeepSeek
    - [ ] Claude
