Code Router 是一个**兼容 OpenAI 接口**的统一 AI 网关（中转）。你只需获取**一个 API 密钥**与**一个统一接入地址**，即可调用多家模型供应商；同时通过模型分组、令牌分组、配额与用量统计，对调用进行集中管理。Code Router 基于开源项目二次开发，源码托管在 [GitHub](https://github.com/ssgzy/Router-new-api)。

<Callout type="info">
  Code Router 目前为全新演示项目，文档中涉及的价格、计费、支付与服务条款等运营信息均为占位说明（待补），请以正式上线后的公告为准。
</Callout>

## 接入地址

所有请求统一发往兼容 OpenAI 协议的接入地址。下方为演示用占位地址，请在实际部署中替换为你自己的域名：

```text
# 基础地址
https://api.code-router.example

# OpenAI 兼容端点
https://api.code-router.example/v1
```

将现有 OpenAI SDK 或客户端的 `base_url` 指向上述端点，并使用你在控制台创建的密钥，即可无缝切换到 Code Router。
