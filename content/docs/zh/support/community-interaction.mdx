欢迎加入 AI Router 开源社区！在这里，你可以与其他开发者交流使用、部署与二次开发经验，分享想法并参与项目改进。

AI Router 是一个统一的、兼容 OpenAI 接口的 AI 网关/中转服务（基于开源项目二次开发）：用户可以使用一套 API Key 和一个 Base URL 调用多家模型供应商，并通过模型分组、令牌分组、配额与用量统计进行管理。

## 社区准则

为了维护良好的社区氛围，参与交流时请遵守以下准则：

<Callout type="warn" title="重要提示">
  社区渠道由志愿者维护，没有义务为你提供一对一的技术支持。遇到问题时，建议优先提交 issue 以便公开追踪和协作解决。
</Callout>

1. 讨论应聚焦 AI Router 的使用、部署、开发与反馈，不发布账号交易、购买引导或求购相关信息。
2. 请通过官方仓库、文档和社区渠道获取项目信息，避免传播未经核实的内容。
3. AI Router 采用 GNU AGPLv3 开源协议，遵守开源协议即可免费使用。详情请查看[项目介绍](/zh/docs/guide/wiki/basic-concepts/project-introduction)。
4. 使用本项目时应遵循相关法律法规、上游服务条款、平台规则和内容安全要求。
5. 社区讨论应围绕合法授权、合规部署和正当工程问题展开。

## 提交一个高质量的 issue

<Steps>
<Step>
搜索现有 issue，确认问题或建议尚未被提出，避免重复。
</Step>
<Step>
描述预期行为与实际行为，并提供最小可复现示例（例如一次对 `https://ai.sammier.com/v1` 的调用）。
</Step>
<Step>
附上必要的环境信息与日志片段，注意去除 API Key、令牌等敏感信息。
</Step>
</Steps>
