## 在反馈中包含的信息

为了让我们更高效地处理你的反馈，提交时尽量附上以下内容：

- 你使用的接口与示例请求（注意脱敏，**不要泄露 API Key**）。
  - OpenAI 兼容端点示例：`https://ai.sammier.com/v1`
- 期望结果与实际结果，以及完整的错误信息或状态码。
- 涉及的模型名称、模型分组或令牌分组等配置信息。
- 复现步骤；如可能，附上能稳定复现的最小示例。
