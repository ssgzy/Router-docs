## 关于本页面

AI Router 是一个统一的、兼容 OpenAI 接口的 AI 网关/中转：你只需一个 API Key 与一个 Base URL，即可调用多家模型供应商，并支持模型分组、令牌分组、配额与用量统计。其 OpenAI 兼容端点形如：

```text
https://ai.sammier.com/v1
```

待首个版本发布后，本页面将按时间倒序列出各版本的变更摘要。在此之前，如需了解最新进展或提交反馈，请前往源码仓库的 Issues 页面。
