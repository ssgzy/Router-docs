## 基本原则（草拟）

以下为通用、供应商中立的可接受使用要点，适用于任何与 OpenAI 兼容的网关服务：

- **遵守法律法规**：用户须遵守所在地及相关司法辖区的法律法规与监管要求。
- **禁止滥用**：不得将本服务用于非法、欺诈、攻击、骚扰、侵权或其他恶意用途，亦不得规避平台的配额、风控或安全机制。
- **尊重上游条款**：经由本服务调用的上游模型供应商各自有其使用条款与内容政策，用户须一并遵守。
- **内容责任自负**：用户对其输入的提示词以及由此生成、传播的内容负责。
- **凭据安全**：用户须妥善保管自己的 API Key 与账户凭据，因泄露或滥用造成的后果由用户承担。
